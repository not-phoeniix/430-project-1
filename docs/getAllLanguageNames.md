## /api/getAllLanguageNames - `GET`, `HEAD`
Gets the names of all languages stored in the dataset.

### Returns

A JSON array of all language names stored.

Format: `[string]`

### Examples
**Request:** `/api/getAllLanguageNames`

**Response:** `["exampleScript", "awfulScript++", ...]`
