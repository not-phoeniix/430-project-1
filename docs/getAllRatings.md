## /api/getAllRatings - `GET`, `HEAD`
Gets all stored rating data.

### Returns
A JSON array of all ratings stored.

### Examples
**Request:** `/api/getAllLanguages`

**Response:** `[{name: "exampleScript", year: 2026, ...}, {name: "awfulScript++", year: 1908, ...}, ...]`
