# Programming Language API Documentation

## Endpoints

### /api/getAllLanguages - `GET`, `HEAD`
**Returns**

A JSON array of all languages stored.

Format: `[{name: string, year: number, creator: string, paradigm: [string], typing: string, logo: string}]`
